## 🔧 API ENDPOINTS

### Authentication
- `POST /api/auth/login` - Login with OAuth2 form
- `POST /api/auth/login-json` - Login with JSON
- `POST /api/auth/register` - Register new user
- `GET /api/auth/me` - Get current user
- `POST /api/auth/refresh-token` - Refresh JWT token
- `GET /api/auth/profile` - Get user profile
- `PUT /api/auth/profile` - Update profile

### Nanolearning
- `GET /api/nano/courses` - List nanolearning courses
- `GET /api/nano/courses/{id}` - Get course details
- `GET /api/nano/modules` - List modules
- `POST /api/nano/progress` - Update progress

### Challenges
- `GET /api/challenges` - List challenges
- `POST /api/challenges/submit` - Submit attempt
- `GET /api/challenges/stats` - Get user stats
- `GET /api/challenges/leaderboard` - Global leaderboard

### Code Sandbox
- `POST /api/sandbox/execute` - Execute code
- `GET /api/sandbox/snippets` - List code snippets
- `POST /api/sandbox/snippets` - Create snippet
- `PUT /api/sandbox/snippets/{id}` - Update snippet

### Search
- `POST /api/search/videos` - Search videos by content
- `GET /api/search/suggestions` - Get search suggestions
- `GET /api/search/filters` - Get available filters

### AR Models
- `GET /api/ar/models` - List AR models
- `GET /api/ar/models/{id}` - Get model details
- `POST /api/ar/qr` - Generate QR code

### Analytics
- `GET /api/analytics/dashboard` - Dashboard statistics
- `GET /api/analytics/heatmap/{video_id}` - Video heatmap
- `GET /api/analytics/students` - Student performance

### Wellness
- `GET /api/wellness/dashboard` - Wellness data
- `POST /api/wellness/session/start` - Start study session
- `POST /api/wellness/breaks` - Start a break

### Video Processing
- `POST /api/videos/transcribe` - Transcribe video
- `POST /api/videos/chapter` - Generate chapters
- `POST /api/videos/extract-resources` - Extract resources
